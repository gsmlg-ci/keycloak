# keycloak

[![Build](https://github.com/gsmlg-ci/keycloak/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/keycloak/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/keycloak`
- `ghcr.io/gsmlg-dev/keycloak`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/keycloak:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/keycloak:latest
```

## Build

```bash
docker build -t gsmlg/keycloak .
```
